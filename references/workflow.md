# Workflow

## Quick HTML

1. Clean input.
2. Write first-principles notes.
3. Write deck outline.
4. Write slide JSON.
5. Generate standalone HTML.
6. Validate with quick-html mode.

## Narrated HTML

Run the quick HTML workflow, then add narration JSON, TTS text files, and optional audio references.

## Composition Output

Use only when requested. Generate the composition HTML, embed a slideshow manifest, align scene identifiers, then validate with the composition mode.
