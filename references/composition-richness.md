# Composition Richness

Avoid single-template monotony. A deck should use a varied visual rhythm while staying coherent.

## Composition rules

- Do not use simple horizontal rows for most content pages.
- Do not use long vertical stacks that overflow the slide.
- Use at least 6 distinct layout types in an 8-12 page deck.
- Adjacent content slides should not use the same layout type.
- Add tags, symbols, numbers, status chips, or relation labels to create information hierarchy.

## Layout vocabulary

- `formula-strip`: equation or decomposition
- `xray-stack`: hidden mechanisms or failure chain
- `flow-chain`: short process with 3-5 steps
- `route-map`: learning route or implementation route
- `diagonal-route`: compact route with diagonal energy
- `hub-spoke`: central concept with branches
- `compact-matrix`: grouped comparison or taxonomy
- `loop-ring`: feedback loop or iteration
- `split-band`: two-level comparison or before/after
- `graph-chain`: dependency or architecture chain
- `constellation`: related signals or concepts in a loose map
- `takeaway-list`: final summary only

## Marker layer

Use symbols semantically:

- CORE, LOOP, STATE, TOOL, EVAL, RISK, CHECK
- !, ?, ✓, ↻, Δ, A/B, 01/02/03

Markers should clarify structure, not decorate randomly.
