# Style Samples

Style samples are design vocabulary references. Do not copy them literally.

## Families

- compact-dark-grid: default compact AI learning deck.
- blueprint-system: system architecture, workflows, Agent engineering.
- executive-radar: reports, strategy, macro and business analysis.
- neon-lab: experiments, AI mechanisms, Reflection and loop concepts.
- warm-risk: risks, constraints, safety and failure modes.
- minimal-evidence: research, evidence chains and dense notes.
- symbolic-circuit: marker-heavy technical concept maps.
- diagonal-route: compact routes and learning paths.
- compact-tag-matrix: dense comparison matrices with tags.

Borrow palette logic, spacing, glow behavior, marker systems, and composition vocabulary. Do not copy exact slide layouts, text placement, or node counts.
